< code >

Use the <code> tag to display code snippets.


< del >

This tag gives text a strikethrough appearance.


< dfn >

According to Merriam Webster, the definition of 'definition' is "a statement of the meaning of a word or word group or a sign or symbol".


< em >

This tag emphasizes text, giving it an italicized appearance.


< font >

*Does not work in HTML5. Use CSS alternative

This sentence should be in a different font.


< i >

This tag gives text an oblique or italicized appearance.


< ins >

This tag indicates that this text or that text was inserted into the document.


< kbd >

The shortcut for opening a folder in VS Code is Ctrl + K Ctrl + O.


< mark >

This tag highlights text, giving it a default yellow background.


< meter >

This tag can be used in many ways:

75%
40 out of 100
20 out of 100

< pre >

      If a text is within the preformatted tag,
      then it is displayed
      in a    preformatted    manner.

< progress >

This tag works like the 'meter' tag. However, it focuses on showing task progress rather than functioning as a gauge (which the 'meter' tag does).

65%
40%
90%

< q >

This tag indicates that a text is a short quotation.

This adds opening and closing quotation marks to the enclosed text.


< rp >, < rt >, < ruby >

These tags are often used for East Asian typography:

可愛い (かわいい)

The text above shows the kanji and hiragana for "kawaii". Here is the code snippet for it:

      <ruby>
        可愛い <rp>(</rp><rt>かわいい</rt><rp>)</rp>
      </ruby>

< s >

This tag also gives text a strikethrough appearance. However, this tag is used to indicate that a text is already outdated or incorrect:

The current year is 2023 2024.


< samp >

This tag indicates or shows text as a sample output from a computer program:

"Hult": Unknown word.

"หนาว": Unknown word.

"Hult": Unknown word.

"kawaii": Unknown word.


< small >

This tag gives text a smaller appearance.


< strike >

*Does not work in HTML5. Use del or s tags as alternative

This tag also gives text a strikethrough appearance.


< strong >

This tag indicates that a text is important, giving it a bold appearance.


< sub >

This tag indicates that a text is subscripted.


< sup >

This tag indicates that a text is superscripted.

< time >

This tag indicates that a text is in a time format:

The current date is February 07, 2024


< tt >

*Does not work in HTML5. Use code, kbd, samp, or CSS alternative

This tag converts text to teletype text.


< u >

This tag gives a text an underlined appearance. However, it is used to indicate that text is unarticulated or styled differently.


< var >

This tag also gives a text an italicized appearance. However, it is used to indicate variables :

a2 + b2 = c2


< wbr >

This tag is used to indicate a possible word break for the browser when a word is waaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaay too loooooooooooooooooooooooooooooooooooooooooooooooooooong.
